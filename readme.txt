
新学期开始了。
要准备好记录学生平时成绩的电子表格。

首先，要得到本学期所教课程的学生名单。
将学生名单保存在文件名中有“学生名单”中文字的电子表格中。
建立相应的空白的电子表格文件，文件名中应该包含下列信息：
1，课程，
2，课程类型，
3，班级。

接着，运行下列程序。
运行时，顺序不能搞错了。
1，clearExcel.py
2, writeNumName.py
3, addTag.py
这样，可以记录学生平时成绩的电子表格就准备好了。
上课时，运行程序4，记录学生的出勤情况，和回答问题，提出问题，课堂作业，作业的情况。
4, mluti_gui-3.py   该文件第一次运行时需要输入电子表格所在的文件夹
记录课堂作业前，要清除上次课堂作业上交情况的记录，以便记录本次的课堂作业上交情况。
每输入一个课堂作业成绩，都会记录该生已经上交了课堂作业。
没有上交课堂作业的学生，会被认为旷课。

当学期结束时，学生的成绩也统计完毕后。运行下面的程序
5，collectMarks.py 用于生成新的文件，将学生的各门成绩集中起来
6, noPassFind.py   用于查找没有及格的学生的门次。
