
新学期开始了。
要准备好记录学生平时成绩的电子表格。

首先，要得到本学期所教课程的学生名单。
将学生名单保存在文件名中有“学生名单”中文字的电子表格中。
建立几个空白的电子表格文件，文件名中应该包含下列信息：
1，课程，
2，课程类型，
3，班级。

接着，运行下列程序。
运行时，顺序不能搞错了。
1，clearExcel.py
2, writeNumName.py
3, addTag.py
这样，可以记录学生平时成绩的电子表格就准备好了。
上课时，运行程序4，记录学生的出勤情况，和回答问题，提出问题的情况。
4, coursemark.py
课后，批改作业时，记录作业成绩，应该运行程序5。
5, writeRecord.py

当学期结束时，学生的成绩也统计完毕后。运行下面的程序
5，collectMarks.py 用于生成新的文件，将学生的各门成绩集中起来
6, noPassFind.py   用于查找没有及格的学生的门次。
